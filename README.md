## How to run
#### Clone repo
``` bash
git clone https://github.com/sukkritsharmaofficial/PyFirebase.git
cd PyFirebase
```
#### Download Dependancies
``` bash
python -m pip install -r requirements.txt
```

#### Run Flask App
``` bash
python app.py
```
